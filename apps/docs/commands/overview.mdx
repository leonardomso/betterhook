## Hidden subcommand

`betterhook __dispatch` is invoked by the installed wrapper script and is
**not** meant for direct use. It's hidden from `--help`.

## Global flags

None yet. Every subcommand has its own flags; see the individual pages.

## Exit codes

betterhook has a [stable exit-code contract](/reference/exit-codes). Agents
can rely on the meaning of `0 / 1 / 2 / 3 / 4 / 5 / 64 / 124 / 130` across
releases.
